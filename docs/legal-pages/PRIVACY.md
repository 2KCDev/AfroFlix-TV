## 2. Données Collectées

Nous collectons les données suivantes :

### Données de Compte
- Email
- Nom d'utilisateur (optionnel)
- Mot de passe (hashé, non stocké en clair)

### Données de Navigation
- Adresse IP
- Type de navigateur
- Pages visitées
- Durée de session
- Source du trafic

### Données de Cookies
- Cookies de session
- Cookies d'authentification
- Cookies Google Analytics

## 3. Utilisation des Données

Vos données sont utilisées pour :
- Authentification et sécurité du compte
- Amélioration du site (analytics)
- Conformité légale et AdSense
- Communication si vous vous inscrivez à la newsletter

## 4. Durée de Conservation

- Comptes utilisateurs : Conservés jusqu'à suppression
- Données d'analytics : 26 mois
- Logs serveur : 30 jours

## 5. Droits RGPD

Vous avez le droit de :
- Accéder à vos données
- Demander leur correction
- Demander leur suppression
- Retirer votre consentement

Pour exercer ces droits : contact@afroflix-tv.com

## 6. Sécurité

Nous utilisons :
- Chiffrement HTTPS
- Hachage des mots de passe (bcrypt)
- Isolation de base de données
- Pare-feu et monitoring

## 7. Tiers Partenaires

- **Google Analytics** : Analytics anonymes
- **Google AdSense** : Publicités (voir Politique de Cookies)
- Aucun partage de données personnelles
