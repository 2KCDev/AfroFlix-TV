# Droits d'Auteur et Signaler un Contenu

## 1. Propriété Intellectuelle

AFROFLIX.TV respecte strictement les droits d'auteur :

- **Textes, critiques, articles** : © AFROFLIX.TV Platform
- **Photos de couverture** : © Studios respectifs
- **Vidéos intégrées** : © Propriétaires officiels (YouTube embeds uniquement)
- **Noms, titres, personnages** : © Propriétaires respectifs

## 2. Contenu Autorisé

Nous n'intégrons que :
- Bandes-annonces officielles YouTube des studios
- Films postés par les studios eux-mêmes
- Contenu avec permission explicite

## 3. Contenu Non Autorisé

Nous INTERDISONS :
- Films pirates
- Liens vers sites de streaming illégaux
- Contenu piraté
- Contenu sans droits

## 4. Procédure de Signalement

Si vous avez détecté un contenu violant les droits d'auteur ou contraire à cette politique :

### Étape 1 : Remplir le formulaire
Accédez à notre formulaire de signalement avec :
- URL exacte de la page
- Description du problème
- Votre nom et email (si applicable)

### Étape 2 : Vérification
Notre équipe examina la réclamation dans un délai de 48 heures.

### Étape 3 : Action
Selon le type de violation :
- **Vidéo piratée** : Retrait immédiat
- **Contenu dupliqué** : Notification à l'auteur + correction
- **Lien illégal** : Retrait immédiat
- **Autre violation** : Évaluation au cas par cas

### Contact Direct
Pour les urgences :
- Email : contact@afroflix-tv.com
- Formulaire : [lien vers formulaire]

## 5. Faux Signalements

Les faux signalements volontaires peuvent entraîner :
- Blocage du compte (si applicable)
- Perte de droits de commentaire
- Signalement à autorités appropriées

## 6. Non-Responsabilité

AFROFLIX.TV n'est responsable que des contenus qu'elle publie directement. Les commentaires utilisateurs sont modérés mais nous n'en sommes pas responsables.
